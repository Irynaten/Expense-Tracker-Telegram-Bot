# Telegram Expense Bot 

A Telegram bot that automatically categorizes uncategorized expenses, sums them up, and calculates the difference between your income and expenses for any chosen period: day, week, or month.  
Provides detailed Excel reports for easy tracking and analysis.

---

## Features
- Auto-categorizes uncategorized expenses
- Calculates total income, total expenses, and balance for day/week/month
- Generates Excel reports
- Simple, fast, user-friendly interface

---

## Commands
| Command | Description |
|---------|-------------|
| `/start` | Shows welcome message and usage example |
| `/income_day` | Income for a selected day |
| `/income_week` | Income for the last week |
| `/income_month` | Income for the last month |
| `/table` | Generate Excel table for the selected period |
| `/cancel` | Delete last user entry |
